### Go Worker Boilerplate

## Features:
1) Dep support
2) Async process (running by process manager)
3) Process signals
4) PID file
5) Parsing flags
6) Docker file from scratch

## Running:
Build docker image:

```bash
make docker-build
```

Build local version:

```bash
make build
```
